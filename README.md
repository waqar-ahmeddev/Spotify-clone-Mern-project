# 🎵 Spotify Clone

This is a Spotify UI Clone built using HTML, CSS, and Bootstrap.

## 🚀 Features

- Responsive Design  
- Sidebar Navigation  
- Music Cards Layout  
- Search Bar  
- Modern UI  

## 🛠 Technologies Used

- HTML5  
- CSS3  
- Bootstrap  
- Google Fonts  
- Font Awesome  

Inspired by Spotify.

## 📂 How to Run

1. Download or Clone the repository:
   git clone https://github.com/waqar-ahmeddev/Spotify-clone-Mern-project.git

2. Open `index.html` in your browser.
